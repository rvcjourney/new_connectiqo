# Connectiqo Website

## Overview
This is the official static website for Connectiqo, a digital product engineering company. The site is built with pure HTML, CSS, and JavaScript, ensuring high performance and zero backend dependencies.

## Features
- **Responsive Design**: Mobile-first approach using CSS Grid and Flexbox.
- **Wave Aesthetic**: Custom wave animations and assets.
- **Smooth Navigation**: Single-page smooth scrolling with active state highlighting.
- **Form Integration**: Ready for Formspree or similar static form backends.
- **Micro-interactions**: Hover effects, scroll animations, and loading states.
- **Animated Logo**: Subtle pulse glow and shine sweep effect.

## Project Structure
- `index.html`: Main entry point.
- `css/`: Stylesheets (`style.css`, `responsive.css`).
- `js/`: Application logic (`main.js`, `form-handler.js`).
- `images/`: Visual assets and icons.
